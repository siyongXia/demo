<h2>解决方案</h2>
- 1.直接让后台给13为的毫秒值:1234567891230
- 2.分别获取时分秒,年月日,来创建日期对象
    ```
    new Date(2017,9,30,14,41,49)
    //但是这样会很坑,创建出来的毫秒值是2017-10-30 14:41:49对应的毫秒值,若想创建2017-09-30 14:41:49,应使用,月份需要-1
    new Date(2017,8,30,14,41,49)
    ```
